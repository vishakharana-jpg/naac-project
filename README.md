# NAAC Self-Assessment Portal - Kumaun University

A web-based NAAC Self-Assessment Portal for Kumaun University, Nainital, Uttarakhand.

## 🚀 Tech Stack

- **Frontend:** React.js, Tailwind CSS, Vite
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Routing:** React Router DOM

## ✨ Features

- Faculty Management (Arts, Science, Management, Technology, Biomedical)
- Department-wise Data Entry Forms
- Grievance Portal Integration
- Sidebar Navigation with Dynamic Content
- Responsive Header with University Branding

## ⚙️ Setup & Installation

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
node server.js
```
